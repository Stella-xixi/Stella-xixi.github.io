CASE SNAPSHOT

项目类型：Data Agent
负责范围：分析规划、代码执行、过程展示、权限控制
当前状态：案例展示页

01 / CHALLENGE

项目背景

业务分析通常依赖专业人员编写查询和制作图表，沟通链路长；同时，模型直接生成代码或 SQL 也可能带来错误结果和安全风险。

02 / SOLUTION

解决方案

Agent 将问题转化为分析计划，在受限环境中执行数据查询与计算，并把工具输入、中间结果和最终图表同步展示给用户。

  • 分析流程：意图理解、指标确认、查询生成与结果解释
  • 透明体验：展示步骤状态、工具输入和中间数据
  • 安全边界：只读权限、查询校验、执行限制与错误拦截
03 / RESULT

项目价值

降低业务分析的使用门槛，同时保留可核验的执行链路；当分析方向有偏差时，用户可以快速定位并修正，而不是只接收一个黑盒答案。
